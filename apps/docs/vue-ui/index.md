# @scxfe/vue-ui

Vue 3 UI 组件库，提供了一系列美观实用的组件。

## 安装

```bash
npm install @scxfe/vue-ui
```

## 快速开始

```vue
<template>
  <Card title="卡片示例">
    <p>卡片内容</p>
  </Card>
</template>

<script setup lang="ts">
import { Card } from '@scxfe/vue-ui';
</script>
```

## 组件列表

- Button
- Counter
- Card
