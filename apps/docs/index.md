# SCX Core

欢迎使用 SCX Core，这是一个基于 React 和 Vue 的组件库和工具集。

## 包概览

- **@scxfe/react-ui**: React 组件库
- **@scxfe/react-hooks**: React 自定义 Hooks
- **@scxfe/vue-ui**: Vue 组件库
- **@scxfe/vue-hooks**: Vue 自定义 Hooks
- **@scxfe/util**: JavaScript 工具集合
- **@scxfe/constants**: 常量定义

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 运行文档应用

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```
